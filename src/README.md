# alt-gpt-v1 Source Code

This directory contains the core source code for the alt-gpt-v1 backend project.

## Structure
- `index.ts`: Entry point for the backend logic
- `@Module.ts`, `Extensions.ts`: Core modules and extensions
- (Planned) `api/`: API endpoint handlers
- (Planned) `llm/`: LLM provider interfaces and implementations
- (Planned) `auth/`: Authentication logic
- (Planned) `history/`: Chat history management
- (Planned) `di/`: Dependency injection setup

For high-level project information, see the root `README.md`.
